## Quick Start

### Prerequisites
- [Obsidian](https://obsidian.md/) (free note-taking app)
- [Claude Code](https://claude.com/claude-code) or Claude with file access
- 5 minutes for setup
- 5-10 minutes daily for journaling

### Installation

1. **Download this repository**
   ```bash
   git clone https://github.com/yourusername/vault-keeper.git
   ```

2. **Copy to your Obsidian vault**
   ```bash
   cp -r vault-keeper /path/to/your/vault/Vault-Keeper-System
   ```

3. **Run setup in Claude**
   Open Claude Code, point it to your vault, and say:
   ```
   Read 'Vault-Keeper-System/AGENT-INSTRUCTIONS.md' and run the setup wizard
   ```

4. **Answer 10 questions** (asked conversationally, 1-2 at a time)

5. **Start journaling** (Claude created your first entry)

**That's it. System is live.**
